## Aplicación consulta API

### Desarrollo

1. **Signal, Resources, rxResources, Señales,Servicios**: entendimiento de los conceptos de Signal, Resources, rxResources, Señales y Servicios en Angular.
2. **Reutilización de componentes, Peticiones http, Operadores de RXJS**: reutilización de componentes, creación de peticiones http con Angular y uso de operadores de RXJS.
3. **Usar un debounceEffect para Buscar el valor tecleado automáticamente**: uso del operador `debounceEffect` para buscar el valor tecleado automáticamente si este cambia, pero no buscar si hace menos de 500ms que no cambia.
4. **Crear un caché para guardar las busquedas anteriores y evitar llamadas a la API**: creación de un caché para guardar las busquedas anteriores y evitar llamadas a la API.
5. **Guardar en localStorage el texto tecleado en las búsquedas para que se pueda recuperar en el siguiente inicio de la app**: guardado en localStorage del texto tecleado en las búsquedas para que se pueda recuperar en el siguiente inicio de la app.
